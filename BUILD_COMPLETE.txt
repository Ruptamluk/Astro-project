================================================================================
                            QUICK START (5 MINUTES)
================================================================================

TERMINAL 1 - Backend:
  cd backend
  python -m venv venv
  source venv/bin/activate       (Windows: venv\Scripts\activate)
  pip install -r requirements.txt
  cp .env.example .env
  python main.py
  
  → Runs on http://localhost:8000

TERMINAL 2 - Frontend:
  pnpm install
  pnpm dev
  
  → Runs on http://localhost:3000

BROWSER:
  1. Open http://localhost:3000
  2. Enter email (demo@example.com)
  3. Check backend console for OTP
  4. Enter OTP
  5. Select birthdate
  6. View your zodiac prediction!

================================================================================
                           ENVIRONMENT CONFIG
================================================================================

FRONTEND (.env.local):
  NEXT_PUBLIC_BACKEND_URL=http://localhost:8000

BACKEND (.env):
  # Required:
  MONGODB_URL=mongodb+srv://user:pass@cluster.mongodb.net/astrology
  
  # Optional (demo mode uses console):
  SMTP_HOST=smtp.gmail.com
  SMTP_PORT=587
  SENDER_EMAIL=your-email@gmail.com
  SENDER_PASSWORD=your-app-password
  
  TWILIO_ACCOUNT_SID=xxx
  TWILIO_AUTH_TOKEN=xxx
  TWILIO_PHONE_NUMBER=+1234567890

See SETUP.md for detailed configuration guide.
